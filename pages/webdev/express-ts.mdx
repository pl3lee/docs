# Express.js with TypeScript
Here is how to create a simple Express.js server with TypeScript.

<Steps>
### Step 1: Create and initialize new project
```bash copy
mkdir server
cd server
npm init -y
```
### Step 2: Install dependencies
```bash copy
npm install express cors
npm install -D @types/express typescript ts-node nodemon @types/cors
```
### Step 3: Create a `tsconfig.json` file
```json copy showLineNumbers filename="tsconfig.json"
{
  "compilerOptions": {
    "target": "es2020",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "outDir": "./dist"
  },
  "include": ["src/**/*.ts"],
  "exclude": ["node_modules"]
}
```
### Step 4: Create a `src` folder and add a `index.ts` file
```ts copy showLineNumbers filename="src/server.ts"
import express from 'express';
import cors from "cors";

export const app = express();
const port = 3001;

app.use(express.json());
app.use(cors());

app.get('/', (req, res) => {
  res.send('Hello, TypeScript with Express!');
});

app.listen(port, () => {
  console.log(`Server is running on http://localhost:${port}`);
});
```
### Step 5: Add scripts to `package.json`
Update the `scripts` section in the `package.json` file to include a `start` script and a `dev` script.
```json copy showLineNumbers filename="package.json"
{
  "scripts": {
    "start": "ts-node src/index.ts",
    "dev": "nodemon src/index.ts"
  }
}
```
### Step 6: Run the server
```bash copy
npm run dev
```
</Steps>
